Sobre mí...

Hola! Soy Jefferson Vivas, un programador Junior especializado en Desarrollo de Aplicaciones Web. Recientemente he completado un Grado Superior en Desarrollo de Aplicaciones Web (DAW) y varios cursos en plataformas como Udemy. A pesar de no contar con experiencia laboral formal, he desarrollado un portafolio sólido de proyectos personales que demuestran mi capacidad para enfrentar desafíos reales en el desarrollo web.

Proyectos

Proyecto de Diseño de Interfaces Web (AMIVET)

Creé un sitio web desde cero, comenzando con la planificación del Site Map, la elaboración de wireframes y el diseño de alta fidelidad utilizando Figma. La implementación se llevó a cabo mediante HTML, dando estilo y formato con CSS y añadiendo transiciones, animaciones, modales y lightbox con JavaScript.

Proyecto Full Stack (Daily-Routine)

Con un grupo de 3 compañeros desarrollamos una aplicación web full stack para la gestión de rutinas diarias. Utilizamos PHP y HTML para crear formularios y páginas dinámicas, y MySQL para el almacenamiento y gestión de datos. Implementamos funciones CRUD (Crear, Leer, Actualizar y Eliminar publicaciones) y utilizamos Bootstrap para el diseño de la interfaz de usuario.
